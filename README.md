# Web Development Project 2 - *Programming-trivia*

Submitted by: **Mark Kamuya**

This web app: **This is a Programming Languages Trivia. It presents trivia questions about programming languages, categorized as easy or hard. Some questions ask users to guess the logo of a programming language, while others test how well one knows the history of programming languages. Users can navigate through the questions, toggle answers, and the background color change based on the difficulty (green for easy, red for hard). The app also allows users to track their progress through the trivia cards.**

Time spent: **5** hours spent in total
